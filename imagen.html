Programas en c#

existe un Top de lenguajes de programación en donde C# (C Sharp) no esté en los primeros lugares. Sin embargo, debes saber que para desarrollar un Software no solo se usa una tecnología, se usan muchas. Para entender el potencial de C# y lo que podemos ser capaces de hacer con ello, sirve citar los grandes proyectos que lo han aprovechado y entender cómo se usan en el día a día. Como una curiosidad, para nombrar este lenguaje los ingenieros de Microsoft se inspiraron en la nota musical C# - un semitono más alto que Do, o C en inglés (el nombre del programa del cual deriva C#). Existen muchas aplicaciones en cuya creación C# ha sido usado solo o en conjunto con otros lenguajes de programación y quiero destacar algunas de las mejores:

Skype para Windows Phone

La ubicua aplicación Skype está disponible en múltiples dispositivos como computadoras Windows, Mac y Linux, móviles, XBox, Smart TVs, etc. Para desarrollarla independientemente en cada plataforma, ha sido necesario emplear el lenguaje de programación más adecuado para cada caso. Por ejemplo, para la versión móvil de Skype en Android se empleó Java,para los clientes iOS Objective C, y C# para la versión móvil en Windows Phone.
